# mosaic-images

## 安装要求
python3.6 +

## 安装

* git clone https://github.com/stars1324/mosaic-images.git

* pip install -r requirements.txt

* 将需要输入的图片改成 input.jpg
* 将需要合成的背景图放到bg目录下
* 执行 python main.py

修改背景图的精细度
> replace_num = 100
